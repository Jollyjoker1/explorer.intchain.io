# INT Blockchain Explorer

> Its frontend is a single page application using vuejs.

> Please visit [https://test.explorer.intchain.io/#/](https://test.explorer.intchain.io/#/) to view all data in INT.

## Run the explorer frontend locally
### Step 1: Install Node.js
1„ÄÅInstall lastest node.js so you can execute 'npm' command by either downloading zip or installer from [https://nodejs.org/en/download/](https://nodejs.org/en/download/) or via [package manager](https://nodejs.org/en/download/package-manager/)

### Step 2: Build and Run

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
